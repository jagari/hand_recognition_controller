🛡️

Intention Gate (의도 분석 필터)

손목과 손가락의 3D 회전 각도를 계산하여 턱을 괴거나 무의식적인 동작을 수행할 때 마우스 제어가 튀는 오작동을 차단합니다.

🚀

Predictive Kinematics (지연 보정)

등속도 물리 연산을 기반으로 약 45ms 앞선 위치를 예측 계산하여, 카메라 추론 및 필터로 인한 지연 피로감을 상쇄합니다.

🎯

Sniper Mode (초정밀 타겟팅)

검지와 엄지가 모여 클릭하는 순간, 미세 횡이동으로 클릭 지점이 밀리지 않도록 좌표 점착 감도(Viscosity = 0.03)를 순간 극대화합니다.

🧠

Neural Drift (실시간 보정)

피로도나 캠 거리 변화에 따라 손가락 펴짐 수치가 달라지면 가우시안 갱신율(LR=0.0005)을 통해 자동으로 경계선을 적응시킵니다.

중앙 백엔드 서비스 아키텍처

1. 고성능 설치 패키지 CDN 배포 스토리지

전용 CDN 및 분산 스토리지 인프라를 바탕으로 대용량의 macOS 및 Windows용 설치 패키지를 중단 없이 신속하고 안전하게 다운로드 제공합니다.

2. Tauri Auto-Updater 버전 관리 API

중앙 자동 업데이트 API 엔드포인트를 운영하여, 기설치된 데스크톱 앱들이 실행 시 신규 버전 릴리즈 여부를 자동 감지해 백그라운드 무중단 업데이트를 진행하도록 제어합니다.

3. 텔레메트리 (성능 지표 및 크래시 로그) 데이터베이스

보안이 강화된 데이터 수집기를 통해 앱 구동 성능지표(손 추적 성공률, 프레임 지연율) 및 크래시 예외 에러 로그를 수집하여 클라이언트 환경을 상시 진단하고 성능을 갱신합니다.
