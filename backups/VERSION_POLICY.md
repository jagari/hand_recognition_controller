# Backup & Versioning Policy

## 1. 버전 체계
- **v1.x.x**: 안정화된 메이저 릴리즈 (2D 모듈화 완료 시점)
- **v0.x.x**: 실험적 기능 도입 중인 상태

## 2. 백업 규칙
1. **성역(Sanctuary) 원칙**: `backups/` 폴더 내의 코드는 직접 수정하지 않는다.
2. **갱신 시점**: 새로운 기능이 "완전 안정화"되고 사용자 승인이 완료된 경우에만 새로운 버전 폴더를 생성하여 복사한다.
3. **롤백 절차**: 실험 중 코드가 망가지면 즉시 최신 `backups/vX.X.X` 폴더의 내용을 루트로 복사하여 복구한다.

## 3. 현재 백업 목록
- **v1.0.0_stable_2d**: [2026-05-18] 2D 기반 정밀 판정, 지수형 브레이크, 로깅 시스템이 통합된 모듈화 버전.
