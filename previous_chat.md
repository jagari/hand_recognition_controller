# Project: AI Vision Hand-Tracking Mouse Controller

## 1. 프로젝트 개요
본 프로젝트는 웹캠 입력(MediaPipe Hand Tracking)을 기반으로 화면의 마우스 커서를 제어하고, 손가락 제스처를 통해 클릭, 더블 클릭, 드래그 앤 드롭을 수행하는 상용 앱 수준의 입력 장치 에이전트이다. 최종 타겟 플랫폼은 macOS 및 Windows 데스크톱 애플리케이션이다.

## 2. 현재 시스템 아키텍처 (Stable 2D Modular)
시스템 안정성과 정밀도를 위해 검증된 2D 픽셀 기반 트래킹 로직을 모듈화된 패키지 구조(`core/`, `backend/`)에서 운용 중이다.

### 핵심 구성 요소 및 흐름:
1. **Camera/Tracker Input**: 멀티프로세싱 기반의 이미지 스트림 처리 및 Hand 랜드마크 추출.
2. **2D Feature Extraction**: 손바닥 크기 대비 픽셀 거리를 기반으로 한 핀치 비율 산출.
3. **Hybrid Control Engine**: 1유로 필터(스무딩), 지수형 브레이크(정밀 조준), 동적 데드존의 결합.
4. **State Machine**: 클릭, 더블 클릭, 드래그 상태를 안전하게 관리.

---

## 3. 내장된 핵심 알고리즘 및 제어 로직

### A. AR/VR 표준 1유로 필터 (One Euro Filter)
* **목적**: 속도 기반 가변 컷오프를 통해 인풋랙 최소화 및 정밀도 극대화.

### B. 지수형 가변 브레이크 (Exponential Variable Brake)
* **목적**: 클릭 시 발생하는 미세 흔들림(Jitter)을 방지하기 위해 핀치 시점에 저항력을 급격히 증가시킴.

### C. 동적 캘리브레이션 (2D Calibration)
* **목적**: 사용자의 손가락 가동 범위에 따른 개인화된 ON/OFF 임계값 설정.
